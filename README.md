ModestWear Ecommerce Website
A full-stack MERN Ecommerce website for shopping abayas, hoodies, watches, shoes, and fashion products.

🚀 Features
User Authentication (Login/Register)
Product Listing
Product Details Page
Add to Cart
Order Placement
Order History
Admin Dashboard
Responsive UI Design
REST API Integration
MongoDB Database

Frontend
React.js

CSS / Bootstrap
Backend
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication
bcryptjs
📂 Project Structure
frontend/
backend/
⚙️ Installation & Setup
Clone Repository
git clone https://github.com/numannkhann029-bot/modestwear-ecommerce.git
Install Frontend Dependencies
cd frontend
npm install
Install Backend Dependencies
cd backend
npm install
Run Backend
npm run server
Run Frontend
npm start
🌐 Environment Variables
Create a .env file in backend folder:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
